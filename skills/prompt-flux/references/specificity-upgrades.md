# Specificity Upgrades

The single highest-leverage move in Flux prompting is replacing generic adjectives with concrete sensory detail. This file is a catalogue of swaps — vague phrase on the left, stronger replacement on the right. Use it as a lookup when you notice yourself reaching for a generic word.

The goal is not to use the *fanciest* specific detail — it's to pick the one specific detail that suits the scene. Pick one. Don't stack three.

## How to use this file

When you've written a Flux prompt and a sentence feels generic, find the offending phrase in this table and pick the replacement that best fits the scene you're building. **Don't stack** — picking one strong specific detail is the move, not three weaker ones in series.

If your scene needs something not covered here, the meta-rule is: replace any adjective with a small concrete fact that *implies* the adjective. "Tired" alone tells the model nothing; "with the dark circles you can't sleep off" gives it something to render.
